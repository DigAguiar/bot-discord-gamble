## Bot for Gamblers

#### Perfect to have fun time w/your friends
I'm using this project to learn more TypeScript, and try to have some fun while building a different project.

---

### Current Features
- **Check Balance:** See your own currency in wallet (`!bal`).
- **Daily Salary:** Collect your daily income once every 24h (`!daily`).

---

### Future Features

#### 1. Core Economy (The Basics)
- **Transfer:** Send money to friends (`!pay @user [amount]`).
- **Inspect Others:** Check other players' balance (`!bal @user`).
- **Leaderboard:** See who are the top 10 richest players (`!top`).

#### 2. The Casino
- **Dice Roll:** High risk, high reward guessing game (`!dice`).
- **Slots:** Visual slot machine with emoji combinations (`!slots`).

#### 3. Crime & Interaction
- **Robbery:** Attempt to steal from a user's wallet (`!rob @user`).
- **Bank Heist:** Group event to rob the server's central bank (`!bankheist`).

#### 4. Progression & Items
- **Shop System:** Buy items that give multipliers or robbery protection (`!shop`).
- **Inventory:** View and manage your owned items (`!inv`).
